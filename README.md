# Vinay Ketha - Software Developer Portfolio

A premium, modern developer portfolio website.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## Folder Structure
- `src/components`: Modular UI sections (Hero, About, etc.)
- `src/constants`: Content and data for the portfolio
- `src/utils`: Helper functions and utilities
- `src/index.css`: Tailwind v4 configuration and global styles
- `index.html`: SEO meta tags and entry point
